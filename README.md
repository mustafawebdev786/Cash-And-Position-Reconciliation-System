# Cash-And-Position-Reconciliation-System
An excel,power BI,Python Data analysis project
Cash & Position Reconciliation System

Excel • Python • Power BI

This project simulates a real financial operations workflow by reconciling internal trade records with broker statements to identify mismatches, missing entries, and account-level discrepancies. It showcases strong practical skills in data cleaning, reconciliation logic, automation, and dashboard reporting.
What This Project Does

Detects matched vs mismatched transactions

Identifies missing records across both systems

Cleans duplicates, blanks, and inconsistent data

Builds a Python-powered reconciliation engine

Visualizes insights using Power BI interactive dashboards

Uses DAX (including COALESCE) to eliminate blank KPIs
Tools Used

Python (Pandas) – Data cleaning & reconciliation logic

Excel – VLOOKUP checks & validation

Power BI – KPIs, slicers, match/mismatch analytics

Key KPIs

Match Rate (%)

Total Mismatches

Missing in Internal / Missing in Broker

Account-Level Risk Flags

Why This Project Matters

This dashboard reflects how real investment firms manage daily reconciliation to prevent financial errors, operational risk, and regulatory issues.
It demonstrates end-to-end data analyst capability across three major tools—ideal for roles in finance, BI, and operations analytics.
